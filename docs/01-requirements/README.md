# Requirements

This folder captures **what the Highland Hideaway web presence must achieve** and under what constraints. It is the source of intent for the implementation described in [`../02-solution/`](../02-solution/).

**Related repository:** the live site source lives in the sibling directory [`highlandhideaway.ca/`](../../../highlandhideaway.ca/) (Hugo static site). This planner repo does not duplicate the application code; it documents requirements and the agreed solution shape.

**Traceability:** when useful, requirement bullets can be tagged inline as `REQ-###` (e.g. REQ-001). Solution docs in `02-solution` may reference those IDs instead of copying full text.
