# Architecture Decision Records (ADR)

This folder holds **ADRs**: short documents that capture a significant decision about the Highland Hideaway web stack or process, its context, and its consequences.

- Background and community practice: [ADR GitHub organization](https://adr.github.io/)
- Requirements live in [`../../01-requirements/`](../../01-requirements/); link ADRs to `REQ-###` IDs when relevant.

## Naming

Use a numeric prefix and kebab-case slug, for example:

- `0001-use-hugo-static-site.md`
- `0002-host-on-github-pages.md`

## Template

Copy [template.md](template.md) when adding a new ADR.
